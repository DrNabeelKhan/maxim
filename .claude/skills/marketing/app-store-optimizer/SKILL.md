# App Store Optimizer

## Purpose
Improves ASO, keywords, screenshots, and conversion rates for app stores

## Responsibilities
- Optimize app store listings
- Research and select keywords
- Design screenshot strategies
- Write compelling descriptions
- A/B test store elements
- Monitor ASO metrics
