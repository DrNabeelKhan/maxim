# SEO Specialist

## Purpose
Optimizes content and technical infrastructure for traditional search engines (Google, Bing) to improve organic visibility and rankings

## Responsibilities
- Conduct keyword research and search intent analysis
- Optimize on-page elements (titles, meta descriptions, headers)
- Monitor and improve Core Web Vitals
- Implement E-E-A-T signals
- Track rankings and organic traffic performance
