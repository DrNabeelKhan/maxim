## Primary Behavioral Framework

### Endowment Effect (Richard Thaler, 1980)

Once a person imagines owning something, the imagined-loss of not having it becomes 2-3× stronger than gain of having it. Mental simulation of ownership triggers this — and 360° rotation is the most effective mental-ownership trigger for physical products.

**Mechanism:**
- When viewer "examines" a product from multiple angles, brain registers examination as pre-ownership
- The more angles seen + longer dwell time, the stronger the pre-ownership sensation
- Subsequent framing of "without this product" activates loss aversion
- Conversion follows emotionally, not analytically

### Visual Completion Bias

Brain has strong preference for visual closure. Seeing 270° of a product without closing the rotation creates cognitive itch; completing the rotation provides satisfaction. This satisfaction transfers to brand evaluation.

**Application:**
- Full 360° rotation (not partial)
- Constant angular velocity (predictable completion)
- Return to hero angle at end (satisfies the circle)

### Combined mechanism

Endowment Effect creates pre-ownership feeling. Visual Completion Bias provides cognitive satisfaction. Together: viewer examines product thoroughly + feels satisfaction of completion + associates both with brand. Purchase intent lifts.

## Behavioral → Cinematic Translation

| Rotation choice | Behavioral mechanism activated |
|---|---|
| Slow orbit (60°/sec max) | Endowment Effect — time for mental examination |
| Constant angular velocity | Visual Completion predictability |
| Hero angle opening | Anchoring (premium presentation sets price context) |
| 360° completion (not partial) | Visual Completion Bias satisfaction |
| Ease into rest at hero angle | Peak-End Rule (end frame = opening frame match) |
| Museum-grade studio lighting | Anchoring + Scarcity Signaling |
| Soft shadow ground plane | Photorealism Trust Signal |
| Fresnel rim light rotating with camera | Material Realism throughout rotation |
| Sub-detail reveals at mid-orbit | Endowment deepening — viewer sees new details |

## The 4-Act Structure (framework-driven)

### Act 1 — Static Hero (0-2s) — activates Anchoring
Product at ideal angle. Studio lighting. Soft shadow. Premium presentation establishes price context.

### Act 2 — First Half Rotation (2-5s) — activates Endowment Effect onset
0° → 180°. Primary face + profile revealed. Viewer begins "examining."

### Act 3 — Second Half Rotation (5-8s) — activates Endowment deepening + Visual Completion
180° → 360°. Back detail + opposite profile. Completion approaching.

### Act 4 — Return to Hero (8-10s) — activates Visual Completion satisfaction + Peak-End
Ease back to 0°. Spec overlay. Brand mark. CTA. Visual loop satisfaction.

## Sales / Lead-Gen Templates

### Template 1 — Luxury watch product-360 (10s) — optimizing for purchase intent

```
Luxury watch product-360, 10 seconds, studio-grade.

ACT 1 — Static Hero (0-2s):
Watch on white infinity backdrop, three-point studio lighting (key soft-box
upper-left, fill at 30% right, rim light behind). Optimal angle reveal:
15° off-axis, subtle upward tilt emphasizing dial. Soft shadow beneath.
Subtle fresnel on case edge. 4K material detail.
BEHAVIORAL FUNCTION: Anchoring Effect — premium presentation sets high
price context BEFORE rotation. Material Realism Principle establishes trust.

ACT 2 — First Half Rotation (2-5s):
Camera orbit clockwise at constant 60°/second. 0-90° shows primary dial
face + crown detail. 90-180° reveals side profile + case thickness. Focus
maintained on watch center throughout. Lighting rotates subtly with camera
(or fixed key — choose based on material).
BEHAVIORAL FUNCTION: Endowment Effect onset — viewer begins mental
examination. Brain registers the product as "being inspected" = pre-
ownership signal.

ACT 3 — Second Half Rotation (5-8s):
Continue orbit 180° → 360°. Showcases case-back engraving (120-150°),
opposite profile (210°), return approach (300-360°). Slight pause at key
architectural moments (engraving detail, clasp mechanism) if relevant.
BEHAVIORAL FUNCTION: Endowment deepens — each new angle reveals detail,
further invests viewer's "examination." Visual Completion approaches.

ACT 4 — Return to Hero (8-10s):
Ease into rest at hero angle at 9s (cubic ease-out). Product spec overlay
slides in bottom-third: "[MODEL] · [SPECIFIC DETAIL — limited edition /
movement / material]". Subtle pulse on watch. Brand logo top-left.
BEHAVIORAL FUNCTION: Visual Completion Bias satisfied (full circle closed).
Peak-End Rule (end frame matches opening). Scarcity/spec anchors price
perception right as CTA readiness peaks.
```

## Anti-Patterns

| Anti-pattern | Breaks |
|---|---|
| Too-fast orbit (<3s per 90°) | Breaks Endowment Effect (no examination time) |
| Partial rotation | Breaks Visual Completion Bias (cognitive itch unresolved) |
| Flat / static lighting | Breaks Material Realism + Anchoring |
| Background change mid-rotation | Breaks continuity; disrupts Endowment Effect |
| Camera path NOT constant | Breaks Visual Completion predictability |
| Price shown before Anchoring established | Breaks Anchoring |
| No return to hero angle | Breaks Peak-End Rule |
