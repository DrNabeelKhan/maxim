## Primary Behavioral Framework

### 2-Second Hook Science

Platforms algorithmically boost videos that hold viewers past 2 seconds. Average scroll threshold: TikTok 1.3s, Instagram 1.8s, YouTube Shorts 2.1s. Past 2s = 3-5× higher completion rate AND 2-3× higher algorithmic distribution.

**Mechanism:**
- Orienting Reflex (physiological): unexpected stimulus triggers involuntary attention
- Pattern break from feed norm = orienting reflex activation
- 2-second window = critical path — must deliver before scroll
- Completion rate compounds — algorithm rewards high-completion with more distribution

### Zeigarnik Effect (Bluma Zeigarnik, 1927)

Uncompleted tasks/thoughts create cognitive tension until resolved. Hooks that open a loop (curiosity gap, teaser reveal, question) demand closure.

**Application:**
- Hook promises payoff (not delivers it)
- First 2 seconds OPEN the loop
- Middle delivers payoff (closes loop satisfyingly)
- End reopens smaller loop (follow/subscribe for more)

### Variable Reward Schedule (B.F. Skinner → Nir Eyal's Hook Model)

Unpredictable reward timing drives stronger engagement than predictable. Content series where each video delivers slightly different reward trains viewer to return.

**Application:**
- Content series where each post follows same hook structure but different payoff
- "Part 1 of 3" framing activates multi-visit behavior
- Trained viewer returns for next variable reward

### Combined mechanism

2-Second Hook captures the scroll-break window. Zeigarnik opens the loop that demands viewing completion. Variable Reward trains the return behavior. Stacked: viewer watches full video + saves + follows + returns for more.

## The 3-Part Social Hook Structure (compressed timing)

### 1. Hook (0-2s) — MANDATORY — activates 2-Second Hook + Zeigarnik + Pattern Interruption
Must stop scroll. Opens a loop. Pattern breaks.

### 2. Payoff (2-90% of runtime) — activates Completion Rate + Variable Reward
Delivers the promised content with sustained engagement. Multi-beat with variable rewards.

### 3. Loop/CTA (final 5-10%) — activates Zeigarnik (reopen loop) + conversion
Close primary loop, open follow-up loop ("part 2 tomorrow" or "follow for more"). Single CTA.

## Sales / Lead-Gen Templates

### Template 1 — TikTok lead-gen hook 15s (optimizing for follow + bio-link click)

```
TikTok-native 15-second vertical lead-gen hook. 9:16. Creator aesthetic.

ACT 1 — HOOK (0-2s):
Creator speaks directly to camera in medium close-up, holds up [OBJECT/
PROP]. Bold text overlay: "The [CATEGORY] mistake 90% of [AUDIENCE] make."
Text enters with 0.2s ease-in-out. Background: neutral interior, natural
light. Handheld subtle shake (platform-native).
BEHAVIORAL FUNCTION: 2-Second Hook — orienting reflex triggers (direct
address). Zeigarnik — "the mistake" opens loop. Pattern Interruption — text
overlay breaks feed visual norm. Target: 85%+ 2-second retention.

ACT 2 — PAYOFF (2-12s):
Three jump cuts showing:
  - Cut 1 (2-4s): the mistake visualized. Whip pan transition in. Text
    overlay names the mistake.
  - Cut 2 (4-8s): the reason behind the mistake. Graphic overlay explains
    the "why" visually.
  - Cut 3 (8-12s): the solution demonstrated. Product/service/insight
    revealed. Creator voice narrates throughout.
Each cut begins with whip-pan transition. Text overlays reinforce key
points. Creator speaking continuously.
BEHAVIORAL FUNCTION: Completion Rate protected (varied pacing sustains
attention). Zeigarnik partially closes (mistake + reason revealed) but
opens sub-loops (solution teased). Variable Reward — each cut delivers
slightly different content.

ACT 3 — LOOP + CTA (12-15s):
Creator close-up: "Follow for the rest of the [CATEGORY] mistakes — 
next one tomorrow." Text overlay "[@handle] → Part 2 tomorrow". Final
frame composition matches first-frame creator position for visual loop.
BEHAVIORAL FUNCTION: Zeigarnik reopens (Part 2 tomorrow) → trains return
behavior. Variable Reward implicit. Visual loop = algorithmic replay bonus.
Follow CTA activated while viewer emotionally primed.
```

## Anti-Patterns (Conversion Killers)

| Anti-pattern | Breaks |
|---|---|
| Hook explains instead of shows | Wastes the 2-second window |
| No loop-back structure | Loses replay bonus |
| Horizontal aspect ratio on vertical-first platform | Immediate scroll |
| Text too small to read at thumb-scroll distance | Accessibility + attention fail |
| Too polished for TikTok specifically | Breaks Platform Native Aesthetic |
| Too much information in 15 seconds | Breaks Completion Rate (overload = drop-off) |
| No sub-loop reopening at end | Breaks Variable Reward training |
| Generic CTA ("check link in bio") | Breaks specificity — no Zeigarnik next step |
| Music mismatched to platform | Breaks Platform Native on TikTok (needs trending or original) |
