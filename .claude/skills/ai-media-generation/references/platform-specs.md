# AI Media Generation — Platform Optimization Specs

Specifications and best practices for each target platform. Every video prompt should be optimized for its destination.

---

## TikTok

| Spec | Value |
|---|---|
| **Aspect Ratio** | 9:16 (vertical, mandatory) |
| **Resolution** | 1080x1920 minimum |
| **Hook Window** | 0.8 seconds — the algorithm measures retention from frame 1 |
| **Optimal Duration** | 15-60 seconds (sweet spot: 21-34s) |
| **Completion Rate Target** | 80%+ for algorithmic boost; below 50% = suppressed |
| **Frame Rate** | 30fps preferred |

### TikTok Prompt Rules
- Pattern interrupt in the FIRST FRAME — not at 2 seconds, at 0.8 seconds
- Trending audio integration boosts reach significantly
- Vertical composition only — no letterboxing
- Text overlays must be within safe zone (avoid top 15% and bottom 20% for UI overlap)
- Loopability: ending that visually connects to beginning increases replay rate
- Quick cuts (0.5-2s per shot) outperform slow transitions
- Completion rate is the single most important metric

## YouTube Shorts

| Spec | Value |
|---|---|
| **Aspect Ratio** | 9:16 (vertical) |
| **Resolution** | 1080x1920 |
| **Hook Window** | 2 seconds — slightly more tolerant than TikTok |
| **Optimal Duration** | Up to 60 seconds |
| **Frame Rate** | 30fps |

### YouTube Shorts Prompt Rules
- 2-second hook standard — more tolerance than TikTok but still critical
- YouTube rewards watch time more heavily than completion rate
- Slightly higher production value expected vs TikTok
- Thumbnails matter less (auto-selected from video) but opening frame IS the thumbnail
- Can lean into educational/informational content more than TikTok
- Subscribe CTA can appear at end without hurting retention

## Instagram Reels

| Spec | Value |
|---|---|
| **Aspect Ratio** | 9:16 (vertical, mandatory for Reels) |
| **Resolution** | 1080x1920 |
| **Hook Window** | 1.5 seconds |
| **Optimal Duration** | 15-90 seconds (sweet spot: 15-30s) |
| **Frame Rate** | 30fps |

### Instagram Reels Prompt Rules
- Aesthetic quality matters MORE on Instagram than any other platform
- Visual polish, color grading, and composition are weighted heavily by algorithm and audience
- Instagram users expect higher production value — avoid raw/unpolished looks unless intentional
- Hashtag strategy still matters for discovery
- Audio: original audio or trending audio both work; original can help with branding
- Carousel-adjacent: content that makes users save or share is boosted
- Text overlays should match brand aesthetic — not just functional

## LinkedIn Video

| Spec | Value |
|---|---|
| **Aspect Ratio** | 16:9 (landscape) or 1:1 (square) — both acceptable. 9:16 works on mobile but not ideal. |
| **Resolution** | 1280x720 minimum |
| **Hook Window** | 2 seconds |
| **Optimal Duration** | 30-120 seconds (sweet spot: 30-90s) |
| **Frame Rate** | 24-30fps |

### LinkedIn Prompt Rules
- Professional tone is NON-NEGOTIABLE — no memes, no TikTok energy
- Insight-driven hook: lead with a surprising stat, contrarian take, or industry truth
- Subtitles/captions are REQUIRED — most LinkedIn video is watched on mute
- 16:9 is preferred for desktop feed; 1:1 is a good compromise for mixed desktop/mobile
- Talking-head or interview format performs well when combined with B-roll
- Value density: every second must deliver insight — no filler
- CTA should be professional (visit site, download report, comment your take)
- Avoid hard-sell — education and thought leadership outperform promotion

## YouTube Standard (Long-Form)

| Spec | Value |
|---|---|
| **Aspect Ratio** | 16:9 (landscape, mandatory) |
| **Resolution** | 1920x1080 minimum (4K preferred: 3840x2160) |
| **Hook Window** | 2 seconds — but first 30 seconds are critical for retention |
| **Optimal Duration** | Any length — 8-15 minutes is algorithm sweet spot |
| **Frame Rate** | 24fps (cinematic) or 30fps (general) |

### YouTube Standard Prompt Rules
- Cinematic quality is expected and rewarded
- Thumbnail is generated separately — but first frame should be compelling
- 2-second hook, then a 30-second "promise" that tells the viewer what they will get
- Longer storytelling arcs are appropriate — use narrative structure
- B-roll, transitions, and color grading all matter for watch time
- SEO: title, description, and tags drive discovery — video content should match
- Chapters (timestamps) improve engagement — structure content with clear sections
- End screens and cards drive subscriptions — plan for them in the edit
