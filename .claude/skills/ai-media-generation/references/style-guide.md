# AI Media Generation — Style Guide

Quick reference for all 15 creative/commercial video styles. Each style has a dedicated external skill with full prompt templates, hook patterns, and genre-specific techniques.

---

## Style Reference

| # | Style | Description | Use When |
|---|---|---|---|
| 01 | **Cinematic** | Film-grade visuals — dramatic lighting, depth of field, anamorphic, lens flares, Steadicam, crane shots | Movie scenes, trailers, brand films, dramatic narratives, any "Hollywood look" request |
| 02 | **3D CGI** | Computer-generated 3D animation — modeling, texturing, volumetric lighting, particle effects | Product visualization, explainers, fantasy/sci-fi, architectural viz, game cinematics |
| 03 | **Cartoon** | 2D animation styles — flat design, cel-shaded, hand-drawn, motion graphics with character animation | Kids content, explainer videos, playful brand content, educational, social media |
| 04 | **Comic to Video** | Panel-by-panel comic art brought to life with motion, transitions, speech bubbles, onomatopoeia | Storytelling, webcomics, fan content, graphic novel trailers, IP adaptation |
| 05 | **Fight Scenes** | High-energy action choreography — martial arts, superpowers, impacts, slow-motion hits | Action trailers, game promos, anime-inspired fights, stunt sequences |
| 06 | **Motion Design Ad** | Animated graphics, typography, shape morphing, kinetic text, brand identity motion | App launches, SaaS demos, event promos, brand identity reveals, UI showcases |
| 07 | **E-commerce Ad** | Product-focused commercial content — showcases, unboxing, lifestyle integration, CTAs | Product launches, Shopify/Amazon ads, TikTok Shop, Instagram Shopping, dropshipping |
| 08 | **Anime Action** | Japanese animation style — speed lines, dramatic poses, exaggerated expressions, cel shading | Anime-style promos, character reveals, fan projects, gaming content |
| 09 | **Product 360** | Turntable rotation, multi-angle showcase, floating product, dramatic reveal | Product pages, launch videos, crowdfunding campaigns, spec highlight reels |
| 10 | **Music Video** | Performance-driven visuals — lip sync, choreography, visual effects, mood-matched editing | Artist promos, album releases, concert visuals, brand music partnerships |
| 11 | **Social Hook** | Scroll-stopping short-form content — pattern interrupts, curiosity gaps, retention mechanics | TikTok, Reels, Shorts, any content where first 2 seconds determine everything |
| 12 | **Brand Story** | Narrative-driven brand content — origin stories, mission films, customer journeys, values | Brand awareness, about-us pages, investor presentations, internal culture films |
| 13 | **Fashion Lookbook** | Outfit showcases, editorial style, model movement, fabric detail, seasonal collections | Fashion brands, lookbooks, seasonal campaigns, influencer content, editorial |
| 14 | **Food & Beverage** | Appetizing visuals — pour shots, steam, texture close-ups, ingredient reveals, plating | Restaurant promos, recipe content, F&B brands, delivery apps, packaging launches |
| 15 | **Real Estate** | Property tours, aerial views, room transitions, lifestyle integration, neighborhood context | Listings, property marketing, architecture showcases, interior design, development promos |

---

## Style Selection Heuristic

If the user does not name a specific style, infer from these signals:

- "Make it look like a movie" → **01-cinematic**
- "3D render" / "CGI" / "animated product" → **02-3d-cgi**
- "Cartoon" / "animated" / "fun style" → **03-cartoon**
- "Comic book" / "panels" / "graphic novel" → **04-comic-to-video**
- "Action" / "fight" / "battle" → **05-fight-scenes**
- "Animated ad" / "motion graphics" / "kinetic text" → **06-motion-design-ad**
- "Product ad" / "e-commerce" / "TikTok Shop" → **07-ecommerce-ad**
- "Anime" / "manga" / "Japanese style" → **08-anime-action**
- "360 view" / "product spin" / "turntable" → **09-product-360**
- "Music video" / "performance" / "lip sync" → **10-music-video**
- "TikTok" / "Reel" / "viral" / "hook" → **11-social-hook**
- "Brand film" / "our story" / "mission" → **12-brand-story**
- "Fashion" / "lookbook" / "outfit" / "editorial" → **13-fashion-lookbook**
- "Food" / "recipe" / "restaurant" / "drink" → **14-food-beverage**
- "Property" / "real estate" / "listing" / "house tour" → **15-real-estate**

When multiple styles overlap, combine the primary style with secondary techniques from the other skill.
