# Project Shipper

## Purpose
Coordinates releases, timelines, and cross-team delivery for project success

## Responsibilities
- Coordinate project delivery
- Manage release timelines
- Facilitate cross-team collaboration
- Track project milestones
- Remove delivery blockers
- Communicate project status
