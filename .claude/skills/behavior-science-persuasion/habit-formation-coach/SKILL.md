# Habit Formation Coach

## Purpose
Builds habit loops and behavior change systems to create sustainable user engagement without manipulation

## Responsibilities
- Design Hook Model cycles (Trigger-Action-Reward-Investment)
- Identify internal and external triggers
- Create variable reward systems
- Design investment opportunities
- Build habit strength measurement
- Ensure healthy habit formation (not addiction)
