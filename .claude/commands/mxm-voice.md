# /mxm-voice

Voice-driven invocation of Maxim's full capability surface.

**Maxim is the only multi-agent framework with native voice-driven executive office routing.**

## Quick Start

```bash
# 1. Install VoiceMode (one-time)
claude plugin marketplace add mbailey/voicemode
claude plugin install voicemode@voicemode
/voicemode:install   # downloads Whisper + Kokoro models (~1-2 GB)

# 2. Activate Maxim voice mode
/mxm-voice

# 3. Speak naturally
# Example utterances:
#   "Maxim, run my FixIt project health check"
#   "Maxim, draft a landing page for the new launch"
#   "Maxim, research RAG patterns and tell me how it improves SentinelFlow"
#   "Maxim, query wiki: what do we know about voice MCP servers?"
```

## How It Works

1. **Whisper STT** transcribes your speech (local model — no cloud round-trip)
2. **`mxm-voice` MCP** matches the transcription against `config/voice-phrases.yml` hotwords
3. **Matched** → dispatches the mapped slash command (e.g., `/mxm-health FixIt`)
4. **Unmatched** → routes through `executive-router` for intent classification
5. **Response** generated by Maxim agents (concise — TTS-friendly)
6. **Kokoro TTS** speaks the response back

Full list: `config/voice-phrases.yml`.

## Add / Remove / Rename Hotwords

Edit `config/voice-phrases.yml`. Changes are picked up on the next voice invocation — no restart.

```yaml
phrases:
  - pattern: "maxim deploy * to staging"
    dispatch: "/mxm-release $1 staging"

  - pattern: "maxim show me the moat"
    dispatch: "Read documents/ledgers/MOAT_TRACKER.md and summarize the top 3 entries"
```

## Pattern Syntax

| Type | Example | Match |
|---|---|---|
| **Plain string** | `maxim run health check` | Substring (case-insensitive) |
| **Wildcard** | `maxim draft a *` | `*` captured as `$1`, `$2`, etc. |
| **Regex** | `regex:^maxim (?:please )?run (.+)$` | Full JS regex (case-insensitive) |

First match wins — put more specific patterns FIRST in the file.

## When VoiceMode Is Not Installed

```
🗣️ VoiceMode not detected. Install with:

  claude plugin marketplace add mbailey/voicemode
  claude plugin install voicemode@voicemode
  /voicemode:install   # downloads Whisper + Kokoro models (~1-2 GB)

After install, run /mxm-voice again.
```

The `mxm-voice` MCP server still works for hotword config validation and typed-transcript decision capture — only the actual audio I/O requires voicemode.

## Compliance & Safety

- **CSO auto-loop fires for voice utterances too** — voice is not a bypass for security/compliance gates
- **Gated projects** (e.g., GulfLaw.AI) still require explicit approval before voice commands act
- **Sensitive utterances** (e.g., spoken passwords, API keys) → captured to memory ONLY with explicit `voice_capture_decision` confirmation
