# /mxm-help

Displays the Maxim command reference card organized by function group. Works in Claude Code, Claude CLI, Claude Desktop.

## Usage
- Claude Code: `/mxm-help`
- Claude CLI: `claude "/mxm-help"`
- Claude Desktop: type `/mxm-help` in chat
