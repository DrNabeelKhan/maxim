# Business Documentation

> Business plans, investor narratives, market research, financial models, and partnership docs.

## Typical Contents

```
documents/business/
├── INVESTOR_NARRATIVE.md       ← Investment thesis and fundraising story
├── BUSINESS_MODEL.md           ← Revenue model, unit economics, pricing
├── MARKET_RESEARCH.md          ← TAM/SAM/SOM, competitive landscape
├── FINANCIAL_MODEL.xlsx        ← Projections, burn rate, runway
├── PARTNERSHIP_AGREEMENTS/     ← Partner docs (may contain sensitive info)
└── PITCH_DECK.pptx             ← Investor presentation
```

## Relationship to CEO Automation

If you've run `/mxm-ceo-setup`, your `.mxm-executive-summary/` folder contains the operational version of many of these documents (CONTEXT.md, METRICS.md, RUNWAY.md, etc.). This `documents/business/` folder is for the canonical, version-controlled business documentation.
