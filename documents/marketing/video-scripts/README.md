# video-scripts/

> Scripts + shot lists for YouTube, Twitter video, Show HN video embeds, and landing page hero videos.

## Naming

`<topic>-<duration>.md`

Examples:
- `install-first-run-10min.md`
- `cso-auto-loop-demo-8min.md`
- `voice-mode-demo-6min.md`
- `agency-walkthrough-15min.md`
- `mxm-vs-landgraph-crewai-12min.md`
- `hero-video-60sec.md`
- `show-hn-embed-90sec.md`

## Script template

1. **Purpose** — what should viewer do after watching?
2. **Target length** — total minutes + segment breakdown
3. **Hook** — first 5 seconds, stop-the-scroll moment
4. **Beats** — timestamped script
5. **Shot list** — which terminals, files, UIs to show
6. **Text overlays** — on-screen labels, quotes, callouts
7. **CTA** — what action at the end?
8. **Thumbnail guidance** — headline + visual + color
9. **Description copy** — YouTube description + tags
10. **Distribution plan** — where this gets posted besides YouTube

## Canonical rules

1. **No stock music that has to clear copyright.** Either original, royalty-free, or silent.
2. **Every claim visible in terminal or UI.** If it's said in voiceover, it also shows on screen.
3. **Voice: DrNabeelKhan narrates.** First-person. Matches `.brand-foundation/personal/voice-profile.md` when it lands.
4. **No AI voiceover.** Authenticity matters on developer YouTube.
5. **Captions are non-negotiable.** WCAG 2.1 AA means accurate captions, not auto-generated.
