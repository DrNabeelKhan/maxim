## Executive summary

Maxim is a **governed multi-agent operating system** that ships as a Claude-native framework with three deployment surfaces:

- **CLI / IDE (100% fidelity):** Claude Code, Cursor, Windsurf, Continue
- **Cowork plugin (85% fidelity):** Cowork environment
- **Web/Desktop Projects (60% fidelity):** Claude Desktop, claude.ai Projects

Core framework is free (BSL 1.1, converts to Apache 2.0 in 4 years). Six premium packs gate specific depth by proprietary license.
